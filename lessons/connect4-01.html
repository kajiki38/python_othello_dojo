コネクトフォー LESSON 01 / 07

盤面を作ろう

コネクトフォーコースへようこそ。この盤面は正方形ではありません——横7列・縦6段。しかも石は「置く」のではなく「落とす」。ひと味違う盤面づくりから始めましょう。

🎯 このレッスンのゴール

  • コネクトフォーのルールをおさらいする
  • 横7×縦6の「長方形」の盤面を2次元リストで表現できる
  • 正方形前提のコードとの違いに気づく

前提: このコースは「Pythonのきほん」(全4レッスン)を終えている前提で進みます。変数・リスト・for・if・関数があやしい人は、先にそちらをどうぞ。

コネクトフォーのルール: 交互に石を「列」に落とす(座標を直接指定できない!)。石は重力で一番下の空きマスまで落ちる。縦・横・斜めのいずれかに自分の色の石を4つ連続で並べたら勝ち。

STEP 1正方形じゃない盤面

オセロは8×8、五目並べは9×9と、他の種目は縦と横のマス数が同じ「正方形」。コネクトフォーは横7列・縦6段の長方形です。定数を2つに分けて管理しましょう。

このサイトの舞台裏: 実はここまでの盤面ビジュアライザは「正方形専用」でした。コネクトフォーの教材を作るために、横幅と縦幅を別々に扱えるよう拡張されています。あなたのコードも同じように「正方形前提」を崩して考える練習をしましょう。

STEP 22次元リストで長方形の盤面を作る

作り方の考え方自体はオセロ・五目並べと同じ「行のリストを積み重ねる」。ただし行の長さ(WIDTH)と行の数(HEIGHT)が別の値になる点だけ注意します。

[EMPTY] * WIDTH で「横7マスぶんの空き行」を作り、それを range(HEIGHT) ぶん(6回)積み重ねる——横と縦、どちらの数を使うべきかを常に意識するのがこのレッスンの一番のポイントです。

[[EMPTY] * WIDTH for _ in range(HEIGHT)] の形はリスト内包表記という書き方です。読み方は「[EMPTY] * WIDTH =空きマス7個の行を、for _ in range(HEIGHT) =6回くり返し作って、1つのリストにまとめる」(_ は「ループの番号は使いません」という合図の変数名)。きほん Lesson 2 の STEP 5 で、forループ版と並べてくわしく説明しています。

STEP 3座標つきで表示する

print_board も横がWIDTH、縦がHEIGHTに対応させます。

join のカッコの中に直接 for を書くのは、リスト内包表記の [ ] を省いた形です。str(i) は数値を文字列にする関数(join は文字列しかつなげないため)。

実行結果の下に、自動でコネクトフォー盤のビジュアルが表示されたはずです。オセロの緑・五目並べの木目とは違う青いフレームのテーマになっていて、石の色も赤(●)と黄(○)に変わっています。データ構造(. ● ○のリスト)は今までとまったく同じです。

board[y] はy行目(1つのリスト)を取り出し、そのリストの [x] 番目がマスの中身——2段階のインデックスです。盤面が「行のリスト」だから、行(y)が先になります。board[x][y] と逆に書くのが定番のミスなので要注意。

やってみよう

board[y][x] を直接書き換えて、一番下の段(y = HEIGHT - 1)に石を3個以上並べるコードを、下の専用エディタに書いてみましょう。石が「浮いて」いたら重力違反です。実行すると自動でチェックされます。次のレッスンで作る「重力」のイメージを先取りできますよ。

解答例を見る 
EMPTY = "."
BLACK = "●"
WHITE = "○"
WIDTH = 7
HEIGHT = 6

def make_board():
    return [[EMPTY] * WIDTH for _ in range(HEIGHT)]

def print_board(board):
    print("  " + " ".join(str(i) for i in range(WIDTH)))
    for y in range(HEIGHT):
        print(y, " ".join(board[y]))

board = make_board()
board[HEIGHT - 1][2] = BLACK
board[HEIGHT - 1][3] = WHITE
board[HEIGHT - 1][4] = BLACK
print_board(board)

HEIGHT - 1(=5)が一番下の段です。コネクトフォーでは石が上から下に落ちるので、ゲームが進むほど下の段から埋まっていきます。

STEP 4まとめ

  • コネクトフォーの盤面は横7×縦6の長方形。WIDTHとHEIGHTを別々の定数で管理する
  • 作り方の骨組みはオセロ・五目並べと同じ「行を積み重ねる」
  • 盤面ビジュアルはゲームごとに見た目のテーマが変わる(データは共通)
