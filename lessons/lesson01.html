STEP 2変数 — データに名前をつける

オセロのプログラムでは「黒の石の数」「今の手番」など、たくさんのデータを扱います。データに名前をつけて保存しておく仕組みが変数です。

ポイントは最後の行の f"..." です。文字列の前に f を付けると、{ } の中に変数や計算式を埋め込めます。これからたくさん使うので覚えておきましょう。

STEP 3リスト — データを並べて持つ

オセロ盤の1行は「8個のマスの並び」です。こういう「並んだデータ」を扱うのがリスト。オセロ作りで一番大事な道具です。

超重要: リストの番号(インデックス)は 0から 始まります。row[3] は「先頭から数えて4番目」。オセロ盤の座標も0〜7で扱うので、この感覚に慣れておきましょう。

リストは書き換えられる

「石を置く」「石をひっくり返す」は、プログラム的にはリストの中身の書き換えです。もう体験してしまいましょう。

" ".join(row) は「リストの中身をスペースでつないで1つの文字列にする」テクニックです。盤面をきれいに表示するのに使います。

STEP 4くり返し — forループ

8マス、8行、64マス…オセロは「同じ処理のくり返し」だらけ。for ループで一気に処理します。

ここで if も登場しました。「もし〜なら」の条件分岐です。for と if の行末には :(コロン)、その次の行はスペース4つで字下げ(インデント)する——これがPythonの文法の核心です。

チャレンジ:白石を数えよう

上のコードに、白石("○")の数を数えて表示する処理を追加してください。さらに余裕があれば、空きマス(".")の数も。

解答例を見る 
white_count = 0
empty_count = 0
for i in range(8):
    if row[i] == "○":
        white_count = white_count + 1
    if row[i] == ".":
        empty_count = empty_count + 1

print(f"白石は {white_count} 個、空きマスは {empty_count} 個")

ちなみに、Pythonには row.count("○") という便利な書き方もあります。今後こちらも使っていきます。
