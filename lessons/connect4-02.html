コネクトフォー LESSON 02 / 06

重力で列に落とす

コネクトフォー最大の特徴です。オセロや五目並べのように「好きな座標に置く」ことはできません。指定できるのは列(x)だけ——石は重力に従って、その列で一番下の空きマスに落ちます。

🎯 このレッスンのゴール

  • ある列の「一番下の空きマス」を探す next_row を作る
  • 実際に石を落とす drop を作る
  • 満杯の列を判定できるようにする

STEP 1「座標を直接指定できない」という制約

これまでの place(board, x, y, color) のように y を自分で決めることはできません。y は盤面の状態から自動で決まるのがコネクトフォーです。まずは「この列は何段目まで石が積まれているか」を調べる部品から作ります。

range(HEIGHT - 1, -1, -1) は「HEIGHT-1から0まで1ずつ減らす」——下から上に向かって調べています。オセロや五目並べのループはほとんど「上から下・左から右」でしたが、重力を扱うここだけは逆向きです。

STEP 2drop関数で実際に落とす

next_rowで落ちる場所を調べて、そこに置く。この2ステップをまとめたのが drop です。

同じ3列目に3回続けて落としたので、石が下から積み上がっているのが分かるはずです。これが「重力」の正体——ただの「一番下の空きマスを探すループ」です。

STEP 3満杯の列を判定する

列が満杯だと drop は None を返します。ゲームループでは「どの列になら落とせるか」の一覧が必要になるので、先に作っておきましょう。一番上の段(y=0)が空いているかどうかを見れば、その列が満杯かすぐ分かります。

[x for x in range(WIDTH) if ...] は「リスト内包表記」。forループでappendしていくのと同じ結果を1行で書けます。「条件を満たすものだけ集める」場面で重宝するので、覚えておくと便利です。

チャレンジ:何段積まれているか数えよう

各列に何個石が積まれているかを数えて、長さWIDTHのリストで返す column_heights(board) を、下の専用エディタで完成させましょう。next_rowの結果から逆算できます(その列が空なら高さ0、満杯ならHEIGHT)。実行すると、空の盤面・積んだ盤面・満杯の列など複数のテストデータで自動チェックされます。

解答例を見る 
def next_row(board, col):
    for y in range(HEIGHT - 1, -1, -1):
        if board[y][col] == EMPTY:
            return y
    return None

def column_heights(board):
    heights = []
    for x in range(WIDTH):
        y = next_row(board, x)
        height = HEIGHT if y is None else HEIGHT - 1 - y
        heights.append(height)
    return heights

board = make_board()
board[5][0] = BLACK
board[5][3] = BLACK
board[4][3] = WHITE
print(column_heights(board))

next_rowが返すyは「次に石が落ちる場所」なので、積まれている高さは HEIGHT - 1 - y。満杯(y is None)なら高さはHEIGHTそのものです。

STEP 4まとめ

  • コネクトフォーは座標ではなく列を指定するゲーム。yは盤面から自動で決まる
  • next_rowで「下から上に向かって」空きマスを探すのが重力の実装
  • dropは「探す→置く」をまとめた関数。満杯ならNone
