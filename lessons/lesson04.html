LESSON 04 / 08

石をひっくり返す

オセロの心臓部、「挟んだ石を裏返す」ロジックを完成させます。このレッスンが本教材の山場。越えれば景色が変わります。

🎯 このレッスンのゴール

  • 「挟める石のリスト」を返す stones_to_flip を完成させる
  • ルールに従って石を置く place を完成させる
  • 実際に石がパタパタ裏返るのを自分のコードで確認する

STEP 1ルールをプログラムの言葉に翻訳する

オセロのルール「相手の石を挟んだら裏返せる」を、コンピュータが分かる手順に分解すると、こうなります。

  1. 置きたいマスから、8方向それぞれに1マスずつ進む
  2. 進んだ先が相手の石なら、リストに記録してさらに進む
  3. 相手の石の列の先に自分の石があれば、記録した石は全部「挟めた」
  4. 先が空きマスや盤の端なら、その方向は挟めない(記録は捨てる)

STEP 2まず1方向だけで考える

いきなり8方向は大変なので、右方向 (dx, dy) = (1, 0) だけで「挟める石集め」を書いてみます。前回の「直進walk」に、記録用リストを足しただけです。

whileの条件が3つ重なっているのがミソです。「盤の中」かつ「相手の石」の間だけ進む。抜けたあと、line に石が入っていて、かつ止まった場所が自分の石なら挟み成立です。

if line and ... の line は「リストが空っぽでない」という意味になります。空のリストはPythonでは「偽(False)」として扱われる——よく使うイディオムです。

STEP 38方向まとめて — stones_to_flip 完成

あとはこれを for dx, dy in DIRECTIONS: で包んで、8方向分の結果を合体させるだけ。オセロエンジンで最重要の関数、完成させましょう。

テスト結果を確認しましょう。初期盤面で黒が(2,3)に置くと(3,3)の白を挟める、(0,0)は何も挟めないので空リスト——期待通りに動いていますか?

STEP 4place — 置いて、裏返す

仕上げです。stones_to_flip の結果を使って、実際に盤面を書き換える place 関数を作ります。挟めない場所なら「置けないよ」と False を返して盤面を触りません。

白石が黒に変わり、置けない場所はちゃんと拒否される。おめでとうございます、オセロのルールがあなたのコードで動きました!

チャレンジ:連続対局

上のコードの続きに手を追加して、黒→白→黒…と交互に3手以上進めてみましょう。「どこなら置けるか」を頭で考えながら座標を選ぶのがコツ。置けない座標を選ぶと False が返ってくるはずです。

進め方の例を見る 
board = make_board()
place(board, 2, 3, BLACK)   # 黒
place(board, 2, 2, WHITE)   # 白
place(board, 3, 2, BLACK)   # 黒
place(board, 2, 4, WHITE)   # 白
print_board(board)

「次にどこへ置けるか」を毎回頭で探すのは大変ですよね。次のレッスンで、それをコンピュータに探させる関数を作ります。

STEP 5まとめ

  • 挟み判定 = 「8方向ループ × 直進walk × 記録リスト」の組み合わせ
  • stones_to_flip は「裏返せる石のリスト」を返す。空リスト = 置けない
  • place は置く+裏返すを実行し、成功/失敗を True/False で返す
