五目並べ LESSON 01 / 06

盤面を作ろう

五目並べコースへようこそ。まずは盤面づくりから。きほんコースで学んだ「リスト」を積み重ねた、「2次元リスト」という考え方が主役です。

🎯 このレッスンのゴール

  • 五目並べのルールをおさらいする
  • 9×9の盤面を2次元リストで表現できる
  • 盤面を作る make_board と表示する print_board を完成させる

前提: このコースは「Pythonのきほん」(全4レッスン)を終えている前提で進みます。変数・リスト・for・if・関数があやしい人は、先にそちらをどうぞ。

五目並べのルール: 交互に石を置き、縦・横・斜めのいずれかに自分の色の石を5つ連続で並べたら勝ち。オセロと違って石をひっくり返す処理がない分、盤面ロジックはシンプルです。

STEP 1盤面のサイズを決める

本物の五目並べ盤は15×15ですが、この教材では見やすさを優先して9×9を使います。ルールも作り方もまったく同じなので、慣れてきたらサイズを変えて遊んでみてください(実行結果の下に出る盤面ビジュアルの自動表示は15×15まで対応しています)。

STEP 22次元リストで盤面を作る

考え方はシンプル。行のリストを、盤面のリストに積み重ねるだけです。

オセロとの違い: 五目並べは初期配置がありません(盤面は全部空っぽからスタート)。だから make_board は [EMPTY] * SIZE を並べるだけで、中央に石を置く処理は不要です。

STEP 3座標つきで表示する

print_board は、行番号・列番号つきで見やすく表示するのがこの道場の流儀。どの種目でも同じ書き方が通用します。

実行結果の下に、自動で五目並べ盤のビジュアルが表示されたはずです。マスを塗るのではなく交点に石を置く碁盤スタイルになっています。中身のデータ(. ● ○ のリスト)はどの種目でも同じ仕組みで、見た目だけがゲームに合わせて変わります。

STEP 4試しに石を置いてみる

まだ「置く関数」は作っていませんが、board[y][x] = 石 で直接書き込むことはもうできます。中央付近に何個か置いて、盤面ビジュアルで確認してみましょう。

やってみよう

下の専用エディタで、盤面を作って黒石を (2, 2) (3, 3) (4, 4) の斜めに、白石を (5, 5) と (2, 6) に置き、最後に print_board で表示してみましょう。座標 (x, y) の読み書きは board[y][x](yが先)でしたね。実行すると自動でチェックされます。

解答例を見る 
EMPTY = "."
BLACK = "●"
WHITE = "○"
SIZE = 9

def make_board():
    return [[EMPTY] * SIZE for _ in range(SIZE)]

def print_board(board):
    print("  " + " ".join(str(i) for i in range(SIZE)))
    for y in range(SIZE):
        print(y, " ".join(board[y]))

board = make_board()
board[2][2] = BLACK
board[3][3] = BLACK
board[4][4] = BLACK
board[5][5] = WHITE
board[6][2] = WHITE   # (x, y) = (2, 6) は y が先!
print_board(board)

黒石が斜めに3つ並びました。次のレッスン以降で作る「5つ並び判定」が完成すれば、これがあと2つで勝ちの形だと自動で見抜けるようになります。

STEP 5まとめ

  • 五目並べの盤面もオセロと同じ「2次元リスト」で表現できる
  • 初期配置がない分、make_board はシンプル
  • 盤面ビジュアルは自動で碁盤スタイルに切り替わる(中身のデータ構造は共通)
